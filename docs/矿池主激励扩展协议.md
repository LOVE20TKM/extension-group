# LOVE20 行动扩展协议之矿池主激励扩展协议（初稿）

## 1. 关于协议

此扩展协议，基于 LOVE20 扩展协议标准，扩展行动参与机制，对指定矿池类行动中的矿池主做激励

- 自动参与：矿池主无需任何额外代币或链上交互，自动参与
- 自动验证：自动验证，无需人工验证提交

## 2. 初始参数

- 激励的矿池类行动

## 3. 激励铸造

矿池主每轮铸造激励，并同步销毁溢出的激励

矿池主理论激励 = 矿池主激励行动总激励 \* 矿池主名下所有矿池的实际可用票数 / 所有矿池的实际可用票数

矿池主实际激励 = 矿池主理论激励 \* 容量修正系数
销毁激励 = 矿池主理论激励 - 矿池主实际激励

容量修正系数 = MIN（矿池主矿池累计容量上限 / 矿池主名下所有矿池已参与代币总量，1）
